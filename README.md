# Atlantic Renewables / Jarvis — Project Audit Trail

This repository is a public, permanent record of decisions made, questions raised and resolved, and project status for the Jarvis build (Atlantic Renewables' replacement for FreshSales CRM + FreshCaller + the Bubble app).

**Purpose:** if anyone — Patrick, Dean, Jack, or a future collaborator — needs to understand what's been decided and why, or what's outstanding, this is the place to look. No login, no invite, no access to the application codebase required.

**What this is not:** this repo contains no application code, no credentials, and no customer data. For the actual codebase, see the private `atlantic-renewables-app` repository (invite-only).

## Contents

- [`STATUS.md`](STATUS.md) — current project phase, what's built, active blockers
- [`DECISIONS.md`](DECISIONS.md) — architectural and business decisions, with rationale
- [`OPEN-QUESTIONS.md`](OPEN-QUESTIONS.md) — questions awaiting an answer, and resolved questions kept for reference

Each file is updated as the project progresses. Git history on this repo **is** the audit trail — every change is timestamped and attributed.
